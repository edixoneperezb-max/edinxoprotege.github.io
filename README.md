## 🛠️ Guía Maestra: Blindaje de Routers (Proyecto EdinxoProtege)
*Asegura el perímetro de tu hogar con estos pasos esenciales:*

1. **Cambio de Credenciales:** Nunca uses `admin/admin`. Crea una contraseña robusta para el panel de administración.
2. **Desactivar WPS:** Es una de las vulnerabilidades más comunes que usan los atacantes para entrar en segundos. **Desactívalo.**
3. **Cifrado WPA3 o WPA2-AES:** Asegúrate de que tu red WiFi use el cifrado más fuerte disponible.
4. **Ocultar SSID:** Aunque no es una medida definitiva, evita que tu red aparezca a simple vista para atacantes casuales.
5. **Filtrado MAC:** Permite que solo tus dispositivos conocidos puedan conectarse.

## 📚 Diccionario de Ciberseguridad (Ruta ISC2)
*Conceptos clave que domino en mi formación:*

### 🏛️ La Tríada CIA
Es el pilar fundamental de toda estrategia de seguridad:
*   **Confidencialidad:** Asegurar que solo las personas autorizadas vean la información.
*   **Integridad:** Garantizar que los datos no sean alterados sin permiso.
*   **Disponibilidad:** Que los sistemas estén activos cuando se necesiten.

### 🔐 Criptografía Aplicada
*   **Cifrado Simétrico:** Una sola llave para cerrar y abrir (Ej. AES). Rápido y eficiente.
*   **Cifrado Asimétrico:** Usa un par de llaves (Pública y Privada). Es la base de la comunicación segura en internet (Ej. RSA).
