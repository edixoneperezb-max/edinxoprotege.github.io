# EdinxoProtege 🛡️

¡Bienvenido! Este es mi espacio dedicado a la **Ciberseguridad** y la protección de la integridad digital.

### 🚀 Sobre este proyecto
Este repositorio forma parte de mi formación técnica, enfocado en educar a la comunidad sobre la seguridad en redes y el uso de herramientas defensivas.

### 🛡️ Áreas de Estudio
*   **Seguridad en Routers:** Protegiendo el perímetro contra ataques.
*   **Criptografía:** Implementación de cifrado simétrico y asimétrico.
*   **Defensa Digital:** Prácticas para evitar el acceso no autorizado.
